# Plataforma de Jogos 🎮

## 📌 Descrição
Projeto antigo desenvolvido em .NET/Blazor com o objetivo de criar uma plataforma simples de jogos.  
Este repositório representa uma fase inicial de estudos e prática em desenvolvimento web com tecnologias Microsoft.

## 🛠️ Tecnologias Utilizadas
- .NET / C#
- Blazor

## 🚀 Funcionalidades
- Cadastro e listagem de jogos.
- Interface básica para interação com os dados.
- Estrutura simples para aprendizado de componentes Blazor.

## 🎯 Objetivo do Projeto
Este projeto foi criado como **estudo inicial** para explorar Blazor e desenvolvimento web em .NET.  
Não segue padrões avançados de arquitetura (como DDD), mas serviu como base para evoluir em projetos mais robustos.

## 📂 Estrutura do Projeto
- **UI** → Interface em Blazor.
- **Camada de dados** → Estrutura simples para persistência/local storage.
